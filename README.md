# Соцсеть на Vue

Демо: [https://alkorotkovv.github.io/social/](https://alkorotkovv.github.io/social/)

Реализован базовый функционал соцсети

## Функционал:
- Главная страница со списком пользователей
- Поиск пользователей
- Страница профиль пользователя
- Возможность сохранить пользователя в избранное

## Стек:
- Vue
- Tailwind
- Pinia
